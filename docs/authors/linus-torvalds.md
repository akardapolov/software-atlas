## Biography

Linus Benedict Torvalds is a Finnish-American software engineer who created the
Linux kernel and Git. While a student at the University of Helsinki in 1991, he
released the first version of Linux as a free operating system kernel. What began
as a personal project grew into one of the most significant collaborative software
efforts in history, powering everything from smartphones to supercomputers.

Torvalds is also known for his direct communication style and for maintaining
the Linux kernel through a meritocratic, distributed development model. In 2005,
when BitKeeper's free license was revoked for Linux kernel developers, Torvalds
wrote Git in approximately two weeks. Git's design was heavily influenced by the
needs of a large-scale distributed project like the Linux kernel.

## Key Contributions

### Linux Kernel (1991)

A free and open-source Unix-like operating system kernel. Torvalds released the
first version (0.01) in September 1991. The kernel became the foundation for
GNU/Linux distributions and later for Android. The Linux development process
itself became a model for large-scale open source collaboration.

### Git (2005)

A distributed version control system created to replace BitKeeper for Linux
kernel development. Torvalds designed Git with three core priorities:
- **Speed** — operations measured in milliseconds even on large repositories
- **Distributed** — every clone is a full repository with complete history
- **Integrity** — content-addressable storage using SHA-1 hashes

Git's object model (blobs, trees, commits, tags) and lightweight branching model
have made it the dominant VCS in software development.

## Influence

### Influenced by

- **Andrew Tanenbaum** — MINIX inspired early Linux architecture
- **Richard Stallman** — GNU project and free software philosophy

### Influenced

- **Junio Hamano** — took over as Git maintainer in 2005
- **D. Richard Hipp** — creator of SQLite and Fossil DVCS
- The entire open source ecosystem — Linux and Git enabled collaborative development at scale
