## Biography

David MacIver is a software engineer and creator of **Hypothesis**, the
de facto standard library for property-based testing in Python.

MacIver has been a leading voice in the testing community, teaching and
advocating for property-based testing through talks, writing, and
open-source work. He has contributed significantly to making property-based
testing accessible and practical for developers.

## Key Contributions

### Hypothesis (2013)

Created Hypothesis, a modern Python library for property-based testing.

Hypothesis innovations over QuickCheck:
- **Strategy-based generation** — explicit, composable data generators
- **Advanced shrinking** — database of shrinking strategies for better minimal examples
- **Stateful testing** — RuleBasedStateMachine for complex systems
- **Good error messages** — clear, reproducible failures with minimised counterexamples
- **Python integration** — seamless pytest integration, fixture support

### Testing Community

MacIver has contributed to the testing community through:
