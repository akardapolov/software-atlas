## Biography

Alain Colmerauer was a French computer scientist who created **Prolog**, one of the first and most influential logic programming languages.

## Key Contributions

### Prolog (1972)

Colmerauer developed Prolog as a programming language based on formal logic. Programs are written as a set of facts and rules, and execution is performed through logical inference.
