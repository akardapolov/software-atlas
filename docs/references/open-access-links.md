## How to Add Papers

1. Verify that paper is open access or author-hosted
2. Download PDF to `library/open-access-papers/`
3. Name as `{author}-{year}-{short-title}.pdf`
4. Add link to this page
5. Create a documentation page in `docs/works/papers/`
