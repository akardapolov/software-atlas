# Back to the Future: The Story of Squeak

| | |
|---|---|
| **Author** | Dan Ingalls (with Alan Kay, Ted Kaehler, John Maloney, Scott Wallace) |
| **Year** | 1997 |
| **Publication** | OOPSLA |
| **Topic(s)** | Smalltalk, open source, Squeak |

## Summary

This OOPSLA paper introduces **Squeak**, an open-source implementation of Smalltalk created by Ingalls and Kay in 1996. Squeak was designed to return Smalltalk to its research roots and make it accessible as a platform for exploration and education.

## Key Ideas

### Return to Simplicity

Squeak was created to:

- **Simplify Smalltalk** - return to core ideas
- **Remove complexity** - eliminate accumulated features
- **Focus on essentials** - objects, messaging, reflection
- **Enable experimentation** - open, modify, explore

### Open Source Philosophy

Squeak embraced openness:

- **Fully open source** - BSD license
- **Community-driven** - contributions from researchers worldwide
- **Transparent development** - design discussions visible
- **No corporate control** - freed from Xerox PARC constraints

### Cross-Platform Implementation

Squeak's design enabled:

- **Portability** - run on any platform with VM
- **Consistent behavior** - same Smalltalk semantics everywhere
- **Image-based persistence** - save entire system state
- **Browser-based development** - explore and modify running system

### Educational Focus

Squeak became popular for education:

- **Learn Smalltalk** - study OOP through exploration
- **E-Toys** - interactive programming examples
- **Morphic UI** - live, scriptable interface objects
- **Active learning** - students see immediate results

## Historical Significance

### Smalltalk Renaissance

Squeak initiated a Smalltalk revival:

- **Modern dialects** - Pharo builds on Squeak
- **Research platform** - papers, experiments, new ideas
- **Community growth** - global Squeak/Smalltalk community

### Open Source Impact

Squeak demonstrated:

- **Research software** - open source accelerates academic work
- **Community development** - many contributors, rapid evolution
- **License freedom** - BSD allowed commercial use

## Legacy

Squeak continues to influence:

- **Modern Smalltalk** - Pharo, Cuis, GNU Smalltalk
- **Educational tools** - Scratch (based on Squeak), E-Toys
- **Live coding** - notebooks, web-based Smalltalks
