## About This Project

Software Atlas is a structured reference on the evolution of software development
thinking — from foundational concepts like lambda calculus (1936) to modern practices
in distributed systems and DevOps.

The goal is to show how ideas connect and influence each other over time.

## Getting Started

1. **Explore by Timeline**: See how concepts evolved chronologically
2. **Follow a Reading Path**: Choose a focused learning sequence
3. **Browse by Topic**: Dive into specific areas of interest
4. **Study a Language**: See how different languages embody ideas
