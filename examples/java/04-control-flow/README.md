# Control Flow in Java

## Overview

Java control flow includes `if/else`, `switch`, loops, `break/continue`,
and (Java 14+) switch expressions. This example targets Java 17+.

## How to Run

```bash
javac Main.java && java Main
```

## Key Concepts

- **`if/else`** — conditional branching
- **`switch`** — multi-way branching (statements)
- **`switch` expressions** — multi-way branching (Java 14+)
- **`for` loops** — classic for, enhanced for
- **`while` loop**
- **`break/continue`** — loop control
- **Labeled breaks** — break from nested loops
