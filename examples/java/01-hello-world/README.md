# Hello World in Java

## Overview

A simple Hello World program in Java.

## Code

```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```

## How to Run

```bash
# Compile
javac HelloWorld.java

# Run
java HelloWorld
```

## Key Concepts

- `public class HelloWorld` - Class declaration, name must match file name
- `public static void main(String[] args)` - Entry point method
- `System.out.println()` - Print to standard output
- Static typing and strong object-oriented design

## Historical Context

Java was created by James Gosling at Sun Microsystems in 1995. Key influences:
- C/C++ syntax
- Smalltalk's pure OOP model
- Write Once, Run Anywhere (bytecode + JVM)
