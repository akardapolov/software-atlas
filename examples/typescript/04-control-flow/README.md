# Control Flow in TypeScript

## Overview

TypeScript inherits JavaScript's control flow: `if/else`, `switch`,
`for`/`while`/`do-while`, `break`/`continue`, and `for-of`/`for-in`.
Type guards refine types within conditional blocks.

## Code

See `main.ts` in this directory.

## How to Run

```bash
tsc main.ts && node main.js
# Or with ts-node
ts-node main.ts
```

## Key Concepts

- **`if/else`** — conditional branching
- **`switch`** — multi-way branching
- **`for`/`while`/`do-while`** — looping constructs
- **`break`/`continue`** — loop control
- **`for-of`** — iteration over iterables
- **Type guards** — `typeof`, `instanceof`, custom predicates
