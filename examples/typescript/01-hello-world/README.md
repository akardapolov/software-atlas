# Hello World in TypeScript

## Overview

A simple Hello World program in TypeScript, a typed superset of JavaScript.

## Code

```typescript
console.log("Hello, World!");
```

## How to Run

```bash
# Compile and run with ts-node
ts-node hello.ts

# Or compile then run
tsc hello.ts
node hello.js
```

## Key Concepts

- Static typing on top of JavaScript
- Type annotations optional (type inference)
- Compile-time type checking
- Compiles to plain JavaScript

## Historical Context

TypeScript was created by Anders Hejlsberg at Microsoft and announced in 2012. Key influences:
- JavaScript (runtime compatibility)
- C# (type system, Anders Hejlsberg's background)
- Java (classes, interfaces)
