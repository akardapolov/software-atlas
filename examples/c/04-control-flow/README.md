# Control Flow in C

## Overview

C control flow: `if/else`, `switch`, `for/while/do-while`, `break/continue`,
and `goto` (exists, but strongly discouraged in modern style).

## How to Run

```bash
gcc -Wall -Wextra -o control main.c && ./control
```

## Key Concepts

- **`if/else`** — conditional branching
- **`switch`** — multi-way branching
- **`for/while/do-while`** — looping constructs
- **`break/continue`** — loop control
- **`goto`** — exists but strongly discouraged
