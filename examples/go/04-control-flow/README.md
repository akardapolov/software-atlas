# Control Flow in Go

## Overview

Go has `if`, `switch`, and a single looping construct: `for` (used as
`while`/`for-each`). It also supports `defer`, `break/continue`, and
labeled breaks.

## How to Run

```bash
go run main.go
```

## Key Concepts

- **`if`** — conditional with optional `else if` / `else`
- **`switch`** — multi-way branching
- **`for`** — all loops (range-based, condition-based, iterator-based)
- **`defer`** — deferred cleanup
- **`break`/`continue`** — loop control
- **Labeled breaks** — break from nested loops
