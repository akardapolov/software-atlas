# C# Examples

Runnable code examples for the [C# language profile](../../docs/languages/csharp/index.md).

Each example is self-contained: one directory, one `README.md` with explanation,
one `Program.cs` file you can run directly.

---

## How to Run Any Example

```bash
cd examples/csharp/<example-dir>
dotnet run
```

No external dependencies required — examples use only the .NET SDK.
