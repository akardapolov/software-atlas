# OOP in C#

## Overview

Demonstrates classes, inheritance, interfaces, structs, and records in C#.

## Code

See [Program.cs](Program.cs).

## How to Run

```bash
dotnet run
```

## Key Concepts

- **Classes** — reference types with single inheritance
- **`virtual`/`override`** — polymorphism
- **Interfaces** — contracts, multiple interface implementation
- **Structs** — value types, stack-allocated
- **`readonly struct`** — immutable value type
- **Records** — immutable reference types with value equality (C# 9.0)
- **`with` expressions** — non-destructive mutation
- **Deconstruction** — break record/tuple into variables
