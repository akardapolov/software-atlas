## Key Concepts

- **Method syntax** тАФ fluent API: `.Where()`, `.Select()`, `.OrderBy()`
- **Query syntax** тАФ SQL-like: `from ... where ... select`
- **Deferred execution** тАФ query not executed until enumerated
- **Standard operators** тАФ `Where`, `Select`, `OrderBy`, `GroupBy`, `First`, `Any`, `All`, `Take`, `Skip`, `Sum`, `Max`, `Average`
- **Pipelines** тАФ chain operators for complex transformations
