# Variables and Types in C#

## Overview

Demonstrates C# value types, reference types, type inference, tuples, nullable types, and ranges.

## Code

See [Program.cs](Program.cs).

## How to Run

```bash
dotnet run
```

## Key Concepts

- **Value types** (`int`, `double`, `bool`, `struct`) — stored on stack, copied by value
- **Reference types** (`string`, `class`, `array`) — stored on heap, copied by reference
- **`var`** — local type inference (C# 3.0)
- **`int?`** — nullable value type (C# 2.0)
- **`??`** — null-coalescing operator
- **Tuples** — lightweight grouping (C# 7.0)
- **`^` and `..`** — index from end and range operators (C# 8.0)
- **`dynamic`** — runtime typing (C# 4.0)
