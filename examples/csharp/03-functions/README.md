# Functions in C#

## Overview

Demonstrates methods, overloading, local functions, expression-bodied members, generics, and parameter options.

## Code

See [Program.cs](Program.cs).

## How to Run

```bash
dotnet run
```

## Key Concepts

- **Method overloading** — same name, different signatures
- **Default parameters** — optional arguments with defaults
- **Named arguments** — specify parameter names at call site
- **Local functions** — nested methods (C# 7.0)
- **Expression-bodied members** — `=>` syntax for single-expression methods
- **Generic methods** — `Swap<T>(ref T a, ref T b)`
- **`params`** — variable number of arguments
- **`ref`** — pass by reference
