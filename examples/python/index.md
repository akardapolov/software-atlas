# Python Examples

Runnable code examples for the [Python language profile](../../docs/languages/python/index.md).

Each example is self-contained: one directory, one `README.md` with explanation,
one or more `.py` files you can run directly.

---

## How to Run Any Example

```bash
cd examples/python/<example-dir>
python3 main.py        # most examples
python3 hello.py       # 01-hello-world
```

No dependencies required for examples 01–08.
Examples 09 (concurrency) and 10 (testing) use stdlib only.
