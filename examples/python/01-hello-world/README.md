# Hello World in Python

## Overview

A simple Hello World program in Python.

## Code

```python
print("Hello, World!")
```

## How to Run

```bash
# Using Python interpreter
python3 hello.py

# Or Python 2 (legacy)
python hello.py
```

## Key Concepts

- No boilerplate required
- `print()` is a built-in function
- Dynamic typing - no type declarations needed
- Emphasis on readability and simplicity

## Historical Context

Python was created by Guido van Rossum and first released in 1991. Key features:
- Multi-paradigm (imperative, OOP, functional)
- Influenced by ABC, Modula-3, C
- Extensive standard library
- Readability-focused design philosophy
