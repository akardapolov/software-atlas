# Functions in Python

## Overview

Python supports first-class functions, closures, higher-order functions,
decorators, `*args`/`**kwargs`, and limited lambda expressions.

## How to Run

```bash
python3 main.py
```

## Key Concepts

- **First-class functions** — functions are objects, assignable to variables
- **Higher-order functions** — `map`, `filter`, `functools.reduce`
- **Closures** — inner functions capture enclosing scope
- **Lambda** — anonymous one-expression functions
- **Decorators** — syntactic sugar for wrapping functions
- **`*args` / `**kwargs`** — variadic arguments
- **Type hints** — optional annotations for parameters and return types
