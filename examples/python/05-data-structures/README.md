# Data Structures in Python

## Overview

Python provides built-in data structures: **lists** (mutable sequences),
**tuples** (immutable), **dictionaries** (key-value mappings), and
**sets** (unique collections). The `collections` module provides
specialised structures like `deque`, `Counter`, `defaultdict`.

## How to Run

```bash
python3 main.py
```

## Key Concepts

- **List** — mutable, ordered sequence, dynamic size
- **Tuple** — immutable, ordered, fixed size
- **Dictionary** — key-value pairs, hash table lookup
- **Set** — unordered collection of unique elements
- **Deque** — double-ended queue (collections module)
- **Heap** — priority queue (heapq module)
- **List comprehensions** — concise list creation
- **Dictionary comprehensions** — concise dict creation

## Historical Context

Python's data structures emphasise readability and built-in functionality over manual implementation. Lists are dynamically resized arrays, dictionaries use hash tables for O(1) lookups. Python 3.7+ added dataclasses for more structured data, and type hints provide optional static analysis.
