# Control Flow in Python

## Overview

Python control flow is expression-friendly and readable: `if/elif/else`,
`for` over iterables, `while`, `break/continue`, exceptions, and pattern
matching (`match`, Python 3.10+).

## How to Run

```bash
python3 main.py
```

## Key Concepts

- **`if/elif/else`** — conditional branching
- **`for`** — iteration over iterables
- **`while`** — while loop
- **`break/continue`** — loop control
- **Comprehensions** — expression-style loops
- **`match/case`** — structural pattern matching (Python 3.10+)
