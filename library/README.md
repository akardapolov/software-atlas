# Local Library

This directory contains local copies of resources for offline reference.

## Copyright Policy

**Important:** This repository only includes materials that are:
1. Freely redistributable (open-access papers)
2. Personal reading notes (original content)
3. Public domain materials

Commercial books and proprietary content are **not** included. Instead, they are:
- Referenced by citation in the documentation
- Linked to official sources or authorized previews
- Accompanied by personal reading notes when applicable

- And more...

## Adding New Materials

### Open-Access Papers

When adding a paper:
1. Verify the paper is freely redistributable
2. Download the PDF
3. Place in `open-access-papers/` with naming convention: `{author}-{year}-{short-title}.pdf`
4. Create corresponding documentation page in `docs/works/papers/`

### Personal Notes

When adding notes:
1. Use existing naming convention: `{work-id}-notes.md`
2. Include your own original content only (no direct copying of copyrighted text)
3. Link to official purchase/preview sources where available
